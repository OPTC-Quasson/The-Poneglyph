## What is The Poneglyph?

**The Poneglyph** is an open-source companion project for **One Piece Treasure Cruise (OPTC)**.

Our goal is simple:

> Help every OPTC player make better decisions through accurate, searchable and interconnected game data.

Instead of searching through Reddit posts, YouTube videos, Discord messages and spreadsheets, The Poneglyph aims to bring everything together in one place.

Whether you're a brand-new player or a veteran with thousands of days played, The Poneglyph is designed to answer questions quickly and accurately.

---

# Our Philosophy

The Poneglyph is **not** just another database.

It is a **knowledge system**.

Instead of simply displaying information, it helps players answer questions like:

* Which ship should I invest 20 Super Cola into?
* Which ship works best with my captain?
* Which teams can I build for this content?
* Which characters remove this debuff?
* What am I missing for Grand Voyage?
* Which units should I level next?

Knowledge is only valuable when it helps you make better decisions.

That is the purpose of The Poneglyph.
